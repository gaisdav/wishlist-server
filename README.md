# wishlist-server


- docker container ls -a - просмотр контейнеров
- docker ps -a - просмотр запущенных контейнеров
- docker compose up -d - запуск контейнеров
- docker compose stop <id> - остановка контейнера
- docker rm <id> - удаление контейнера
